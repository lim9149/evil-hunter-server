# 04. 원클릭 테스트 설명

## 가장 쉬운 명령어
```bash
python scripts/run_easy_checks.py
```

이 명령은 아래를 순서대로 검사합니다.
1. pytest 자동 테스트
2. FastAPI TestClient로 핵심 API 스모크 테스트
3. 결과를 `reports/easy_check_report.json` 으로 저장

## 성공하면
- 터미널에 PASS가 보입니다.
- `reports/easy_check_report.json` 파일이 생깁니다.

## 실패하면
- 어떤 단계에서 실패했는지 바로 출력됩니다.
- 그 이름을 시트 `작업로그` 탭에 적고 수정 작업을 시작하면 됩니다.
