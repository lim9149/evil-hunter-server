# 05. AI에게 바로 붙여넣는 프롬프트

## 새 채팅방 기본 컨텍스트 전달용
이 프로젝트의 현재 가칭은 **무림객잔: 문파 재건기** 입니다.
핵심은 저사양 모바일용 데이터 기반 운영 RPG이며,
객잔(마을) 안에서 최대 20명의 헌터가 생활 루프를 돌고,
사냥 중 HP / 허기 / 기력이 떨어지면 자동으로 마을 시설로 복귀합니다.

스토리 방향은 최근 인기 무협 장르의 감성을 참고하되,
특정 작품을 직접 복제하지 않는 오리지널 구조입니다.
- 몰락 문파 재건
- 객잔을 거점으로 한 세력전
- 숨겨진 비전/장부/잔향을 읽는 능력
- 초반 15분을 끌어주는 길라잡이 퀘스트 다수 배치

수익화 원칙은 아래를 기본값으로 사용합니다.
- 인앱결제 의존 구조보다 **선택형 보상광고 우선**
- **배너 광고 금지**
- **강제 전면 광고 금지**
- 광고 신전 / 광고 상자 / 광고 패스 / 광고 던전 / 커뮤니티 홍보 슬롯 중심
- 월 200만원+ 목표를 염두에 두되 잔존율을 해치지 않는 방향 우선

법/운영 원칙은 아래를 기본값으로 사용합니다.
- 확률형 보상이 있으면 **게임 내 표기 + 서버 응답 + 시트**를 같이 갱신
- 한국 서비스 기준의 공지/확률표기/변경이력 문구를 남김
