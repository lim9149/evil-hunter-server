# AI 유지보수 운영 가이드

현재 방향은 `객잔 운영 무협 타이쿤`이다.
앞으로 AI에게 작업을 맡길 때 아래 원칙을 지킨다.

1. TownWorldScene 단일 씬 우선
2. 마을에서 헌터 이동, 전투, 귀환이 보이는 구조 유지
3. 스토리/우편/공지/확률표기는 오버레이 UI
4. 광고는 강제가 아니라 선택형이며, 휴식 지점에서만 제안
5. 서버는 보상/진행을 판정하고 Unity는 표현과 조작감을 담당

작업 요청 예시
- TownWorldScene용 hunter prefab 연결
- monster spawn zone 추가
- 시설 앵커와 NavMesh 연결
- TownHUD 오버레이 버튼 연결
- 광고 신전 버튼을 쉬는 구간에만 노출
