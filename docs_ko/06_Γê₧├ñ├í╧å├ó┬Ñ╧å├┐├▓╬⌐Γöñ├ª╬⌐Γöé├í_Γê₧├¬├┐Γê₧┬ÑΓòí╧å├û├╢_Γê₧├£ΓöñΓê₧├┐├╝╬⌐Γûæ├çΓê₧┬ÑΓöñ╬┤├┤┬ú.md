# 06. 선택형광고 수익화 운영 가이드

## 절대 원칙
- 광고는 **플레이어가 원할 때만** 본다.
- **배너 광고 금지**
- **게임 중간 강제 전면 광고 금지**
- 광고 보상은 플레이 루프와 연결한다.
- 하루 캡, 서버 멱등 처리, 악용 방지 로그를 함께 설계한다.

## 우선 구현 순서
1. 광고 신전: 광고 1회 -> 골드/가속/입장권 보상
2. 광고 보물상자: 광고 1회 -> 랜덤 자원 상자 지급
3. 광고 패스: 일일 누적 시청 횟수에 따라 단계 보상 지급
4. 광고 던전: 광고 시청 시 추가 입장권 1회 지급
5. 커뮤니티 홍보 게시판: 게임 내 커뮤니티/디스코드/이벤트 홍보 슬롯

## UX 원칙
- 광고를 보기 전 `이 광고는 선택 사항입니다` 문구를 둔다.
- 보상 버튼은 전투 종료, 마을 복귀, 일일 루틴 시점 같은 **자연스러운 구간**에서만 노출한다.
- 광고를 거절해도 플레이 템포가 무너지지 않게 만든다.
