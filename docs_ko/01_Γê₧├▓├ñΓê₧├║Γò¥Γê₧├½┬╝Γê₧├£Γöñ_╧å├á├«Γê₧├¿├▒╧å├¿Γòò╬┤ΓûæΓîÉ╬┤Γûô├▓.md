# 01. 아주 쉬운 테스트 방법

## 준비
1. 압축을 풀고 `server_source` 폴더를 엽니다.
2. 폴더 안 빈 공간에서 터미널을 엽니다.
3. 아래 순서대로 그대로 입력합니다.

### Windows
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
pytest
uvicorn main:app --reload
```

### macOS / Linux
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
pytest
uvicorn main:app --reload
```

## 정상 확인 1: 자동 테스트
- `pytest` 실행 후 전체 통과가 나오면 기본 로직은 정상입니다.
- 목표 결과: `17 passed`

## 정상 확인 2: 서버 헬스체크
브라우저에서 아래 주소를 엽니다.
- `http://127.0.0.1:8000/health`

정상이라면 상태와 버전 JSON이 보입니다.

## 정상 확인 3: 아주 쉬운 수동 점검
1. `/docs` 페이지를 엽니다.
2. `POST /auth/guest` 실행
3. 발급된 access token으로 hunter 관련 API 테스트
4. hunter 생성 / 조회 / 전투 / 보상 흐름을 한 번 확인
