# AI_GAME_PROJECT_RULES

This project is developed with AI collaboration.

Key rules:
1. Do NOT copy Evil Hunter Tycoon assets, code, UI, names, or data.
2. Only the general gameplay loop inspiration is allowed (town → hunters → battle → rewards → growth).
3. All systems must be newly implemented.
4. Code must include comments so future AI agents can understand it.
5. When changing systems, update documentation.
6. Maintain modular Unity-friendly architecture.
